- react区别
    1. vue语法更多，对新手更友好
        v-指令 @开头的事件和修饰符
    2. 固定的结构
        data(){} 状态
        method()添加事件
        - react不用this就是在一个函数里面不存在this丢失
        - react直接写在函数中，不用额外写method，更加简洁
    3. 状态可以直接修改不需要调用setstate
    4. 动态类名输出
        - react 绑定一个数据
        - vue 有:语法  里面的代码返回结果
    5. vue 提供了计算属性
        - 专门的语法区域 methods函数，结果是要绑定的计算属性
